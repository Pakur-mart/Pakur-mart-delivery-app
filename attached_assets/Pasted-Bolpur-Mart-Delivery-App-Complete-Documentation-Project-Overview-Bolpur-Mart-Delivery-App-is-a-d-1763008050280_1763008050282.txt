Bolpur Mart Delivery App - Complete Documentation
Project Overview
Bolpur Mart Delivery App is a delivery management system for the Bolpur Mart food ordering platform. This application enables delivery personnel to receive orders, update delivery status in real-time, collect cash on delivery payments, and manage their delivery workflow efficiently.
Repository: https://github.com/cehpoint-official/bolpur-mart-delivery-app.git
System Architecture
The application follows a client-server architecture with Firebase as the backend service:
Frontend: React + TypeScript + Vite


Backend: Firebase (Firestore, Authentication, Cloud Functions)


State Management: Zustand


Styling: Tailwind CSS


Real-time Updates: Firebase Firestore listeners


Data Flow
text

                                 Cash Collection
                                      ↓
                              Payment Verification

Features
Core Functionality
1. Order Management
View assigned orders in real-time


Filter orders by status (Pending, Out for Delivery, Delivered)


Order details including customer info, items, address, and total amount


2. Delivery Status Updates
Pending: Order assigned to delivery personnel


Out for Delivery: Order picked up and in transit


Delivered: Successfully delivered to customer


Real-time sync across all platforms (Admin, Customer, Delivery)


3. Cash on Delivery (COD) Management
Mark payment received upon delivery


Track cash collected per delivery


Daily cash collection summary


Payment verification workflow


4. Authentication
Secure login for delivery personnel


Role-based access (delivery-only access)


Session persistence


5. Delivery History
View past deliveries


Track earnings and completed orders


Performance metrics


6. Profile Management
Update personal information


Contact details


Availability status
